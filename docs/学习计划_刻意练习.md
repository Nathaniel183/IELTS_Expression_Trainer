# 刻意练习计划

## 每日 30 分钟

1. 5 分钟：复习昨天的三个表达。
2. 12 分钟：做 6—10 道拼句题。
3. 8 分钟：完成一道自由表达题。
4. 5 分钟：复制 AI 评判提示词，修改回答。

## 四阶段循环

### 1. 识别

先判断题目要求的主要交际功能：描述、叙事、解释、个人表达、论证、解决问题、预测、互动或澄清。

### 2. 拼接

关注语块的功能，不要只看单词：

- structure：句型、连接和语法结构；
- concept：主题内容和具体概念。

### 3. 迁移

做完后替换一个核心概念，把同一句型用于另一个主题。

### 4. 输出

自由表达时先追求自然和清楚，再追求复杂度。每次反馈后至少重写一次。

## 每周安排

- 周一：客观描述与过程。
- 周二：个人状态与经历。
- 周三：概念解释与举例。
- 周四：观点、理由和评价。
- 周五：问题、方案和预测。
- 周六：互动沟通与表达修复。
- 周日：随机综合与复盘。
