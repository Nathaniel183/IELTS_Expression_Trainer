# 数据格式说明 v2.0

本替换包包含以下文件：

```text
data/templates.txt
data/themes.txt
data/questions.txt
data/prompts.txt
data/knowledge_base.txt
assets/js/fallback-data.js
docs/学习计划_刻意练习.md
docs/数据格式说明.md
```

## 1. templates.txt

通用句型模板库。核心结构：

```json
{
  "version": "2.0",
  "templateGroups": [
    {
      "id": "position",
      "functionName": "表明立场",
      "purpose": "用于表达个人观点或总体立场。",
      "templates": [
        {
          "id": "tpl-pos-001",
          "text": "In my view, [opinion].",
          "zh": "在我看来，[观点]。",
          "slots": ["opinion"],
          "level": "easy"
        }
      ]
    }
  ]
}
```

## 2. themes.txt

主题库。每个主题包含：

- `id`
- `name`
- `icon`
- `description`
- `coreVocab`
- `commonClaims`
- `practiceSemantics`

其中 `practiceSemantics` 是为了生成题目和提示词而设计的语义字段，例如：

```json
{
  "topic": "online and school education",
  "reason": "it helps students develop skills that are useful in real life",
  "solution": "combine classroom teaching with high-quality online resources"
}
```

## 3. questions.txt

题库。本版共 672 道题。

### build 题

```json
{
  "id": "education-build-001",
  "themeId": "education",
  "type": "build",
  "tokens": [
    {"id": "a", "text": "In recent years,", "kind": "template"}
  ],
  "acceptableAnswers": [["a", "b", "c"]]
}
```

### free 题

```json
{
  "id": "education-free-001",
  "themeId": "education",
  "type": "free",
  "suggestedTemplateIds": ["tpl-pos-001"],
  "suggestedVocabIds": ["education-v001"],
  "sampleAnswer": "In my view, ..."
}
```

## 4. fallback-data.js

这是给“直接双击 html 文件打开”准备的内置备用数据。

如果你只替换 data/*.txt，不替换 fallback-data.js，那么：
- 用本地服务器运行时会读新数据；
- 直接双击打开时可能仍显示旧备用数据。

所以本替换包同时提供了新的 fallback-data.js。
