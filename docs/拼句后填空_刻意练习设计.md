# 拼句后填空的刻意练习设计

## 一、为什么不能只做选择

拼句选择能够训练：

- 识别正确语块；
- 判断语序；
- 区分结构词组与主题词组；
- 熟悉完整句子的整体形式。

但选择题提供了较强提示。学习者可能只是在候选项之间进行比较，并不一定能从中文意义主动提取英文表达。

## 二、为什么加入填空

填空要求学习者完成：

```text
中文提示 → 主动提取英文词组 → 输入 → 检查
```

它减少了选项提示，直接训练中文和英文核心语块之间的联系。

## 三、挖空原则

每题只挖一至三个核心表达：

1. 当前题型最重要的句型结构；
2. 高频搭配；
3. 主题核心概念；
4. 礼貌请求、让步、反驳、条件和交流修复等功能语块。

不会挖空：

- 无实际学习价值的冠词或普通代词；
- 过多连续内容；
- 整个答案句；
- 只考机械拼写而不考表达的片段。

## 四、难度控制

- 短句：通常一个空；
- 中等句：通常两个空；
- 多句回答：最多三个空；
- 每个空通常是一个完整词组；
- 中文提示始终保留；
- 提示按钮只显示词数和首字母。

## 五、反馈流程

1. 填错后显示错误空位，但不立即显示答案；
2. 学习者可以再次尝试；
3. 必要时查看词数和首字母；
4. 最后才显示答案；
5. 显示答案后必须清空并重新独立填写，才能计为掌握。

## 六、建议复习

第二天优先筛选或重做“◐ 待填空”的题。反复填错的表达应在另一个主题中重新造句。
