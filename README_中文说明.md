# IELTS Expression Trainer 中文说明

这是一个本地静态网页练习项目，目标是把雅思写作与口语表达知识库转化为可重复训练的题库。

## 项目目标

它不是单纯背模板，而是按“刻意练习”设计：

1. 明确目标：每道题只训练一个或几个表达点。
2. 降低负担：拼句题不要求打字拼写，重点练结构选择和语义组合。
3. 即时反馈：拼句题可立即判断。
4. 输出反馈：自由表达题可复制 AI 评判提示词。
5. 重复迁移：同一批核心句型在不同主题中反复出现。
6. 记录复盘：自由表达题会保存历史回答。

## 如何运行

推荐方式：

```bash
python start_server.py
```

或：

```bash
python -m http.server 8000
```

然后打开：

```text
http://localhost:8000
```

也可以直接双击 `index.html`，项目内置备用数据，基本功能仍可使用。但如果你修改了 `data/*.txt`，推荐用本地服务器运行，否则浏览器可能因安全限制无法读取本地 txt 文件。

## 两类题型

### 1. build 拼句选择题

题干给出中文含义，你点击或拖拽英文片段拼出完整句子。

训练目标：

- 熟悉通用句型模板。
- 熟悉主题词汇与句型组合。
- 减少拼写负担，把注意力放在结构和语义。
- 允许多种正确答案，通过 `acceptableAnswers` 设置。

### 2. free 自由表达题

题干给出写作或口语任务，你自行组织英文句子或段落。页面提供模板和主题词汇，点击可插入输入框。

训练目标：

- 从识别句型过渡到主动输出。
- 练习句子、段落、口语回答、微型作文。
- 保存历史回答。
- 复制 AI 评判提示词，让 AI 根据题目和回答给反馈。

## 添加主题或题目

主页提供提示词复制按钮：

1. 生成新主题数据。
2. 生成新题目数据。
3. 修改整个项目。
4. 生成 CSS 视觉主题。

推荐流程：

1. 复制提示词。
2. 填写想新增的主题或需求。
3. 让 AI 输出 JSON。
4. 合并到 `data/themes.txt` 或 `data/questions.txt`。
5. 用本地服务器重新打开。

## 换肤方式

样式集中在：

```text
assets/css/style.css
```

最简单是替换 `:root` 中的变量：

```css
:root {
  --bg: #f6f7fb;
  --surface: rgba(255,255,255,.88);
  --text: #172033;
  --primary: #4f46e5;
}
```

主页有 CSS 换肤提示词，可让 AI 生成新主题。

## 进度保存

进度保存在浏览器 `localStorage`：

- 拼句题：是否完成、尝试次数、最近是否正确。
- 自由表达题：历史回答。
- 主页：总题数、已完成数、表达记录数。

不会上传任何服务器。清除浏览器数据会丢失进度。
